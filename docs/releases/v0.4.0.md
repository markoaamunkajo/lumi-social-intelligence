# Lumi Social Intelligence v0.4.0: review-gated Live Surface natural-language controls

v0.4.0 is the real public release of review-gated Live Surface natural-language controls.

This is not a shadow-only release. The public API now produces review-gated control cards for safe session readiness state and fails closed when a requested personal-data surface is not available.

## Observed / verified in this repository

- Real review-gated public capability
- `shadow_only: false`
- Semantic Live Surface intent parsing
- Safe session-readiness review cards
- Missing Calendar behavior: `Calendar doesn't yet exist on Live Surface, would you like to add it?`
- Review-gated / no-write boundary
- `canonical_writes: 0`
- `external_writes: 0`
- `private_runtime_reads: 0`
- `scheduler_mutations: 0`

## Claim boundary

This release claims real review-gated public Live Surface controls. It is not autonomous private runtime automation.

## Not claimed

- Telegram sends or reactions
- Calendar/email reads without explicit reviewed surface membership
- Runtime config or scheduler mutation
- Durable memory writes
- Raw private Hermes runtime state

## Public API

```python
from lumi_social_intelligence import apply_review_gated_control
```

The API returns a review card and zero side-effect counters. Safe session-readiness controls can be applied by a host public API; personal-data requests fail closed until a surface is explicitly added and reviewed.
