# Lumi Social Intelligence v0.5.0

**Status:** public release
**Release theme:** cache-backed fast lane for Live Surface—instant only when it is local, authorized, bounded, and safe to skip agent work.

## Summary

v0.5.0 defines a public-safe fast-lane contract for a **private host adapter**. It can promote an approved tool capability to Live Surface, allowing a previously refreshed local cache to serve a tiny, immediate result without opening the agent/tool path only when every precondition is satisfied:

- the sender is authorized;
- the message is a Telegram direct message;
- the capability is explicitly host-approved for Live Surface promotion;
- a local cache is both valid and no older than the **12-hour cache freshness ceiling**; and
- normal adapter delivery remains available.

The fast lane is not a general bypass. It performs no network I/O, opens no agent session or compression path, permits zero agent tool calls, and limits the reply to 4096 characters.

## Fail closed

**Falls through normally:** any uncertainty returns to ordinary handling:

| Condition | Required behavior |
|---|---|
| Unauthorized sender | Normal authorization/pairing flow |
| Non-Telegram or non-DM traffic | Normal full dispatch |
| Missing or unapproved Live Surface capability | Normal full dispatch |
| Missing, invalid, or stale cache | Normal full dispatch |

**Safety order:** authorization precedes direct reply. That order is deliberate: the quick path must never widen who can receive a response or what data it may use.

## Observed / verified in this repository

- `build_cache_backed_fast_lane_plan` emits the deterministic, public-safe v0.5.0 host contract.
- The contract scopes the lane to `telegram_direct_message` and a host-approved Live Surface capability.
- The cache contract records a host-local-only cache, no network I/O, no raw-cache export, and a 43,200-second freshness ceiling.
- The dispatch contract records authorization before reply, no agent session/compression, zero tool calls, a 4096-character cap, and normal adapter delivery.
- Every failed predicate resolves to a named normal-dispatch fallback.
- Public evidence is generated in `docs/evidence/v0.5.0-cache-fast-lane-evidence.json` and `.md`.

## Review-gated / no-write boundary

This public release is contract and evidence—not the private runtime implementation. It does **not** ship the private Hermes runtime adapter, read local cache content, include private chat identifiers, send Telegram messages, or promote a runtime surface.

```text
canonical_writes: 0
telegram_messages_sent_by_public_repo: 0
telegram_reactions_sent_by_public_repo: 0
private_runtime_reads_by_public_repo: 0
runtime_mutations_by_public_repo: 0
```

## Not claimed

- The public repository can read a host cache or private runtime state.
- The public repository sends a Telegram reply.
- The fast lane is available without host approval, to groups, or to unauthorized people.
- A stale, invalid, or missing cache may be used for an instant reply.
- The fast lane replaces normal agent dispatch outside its narrow eligibility boundary.
