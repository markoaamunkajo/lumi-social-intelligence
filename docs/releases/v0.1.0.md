# Lumi Social Intelligence v0.1.0

First real **v0.1.0** release for **Lumi Social Intelligence**: the public doorway for **Lumi Layered Memory**, **Nuances**, **Presence**, and the **Lumi for Hermes** preview.

## Positioning

> We are making the self-improvement loop something that is human-shaped.

**Lumi Social Intelligence** is a governed reflection layer for agent self-improvement. It helps an agent reflect on memory, nuance, and initiative without turning every interaction into an uncontrolled identity rewrite.

This release presents Lumi as **not an auto-personality-rewriter**. Self-improvement proposals should be inspectable, confidence-scored, contradiction-aware, reviewable, and reversible. Agents should propose, explain, and ask before durable changes.

## What is included

- **Presence** public skeleton source, including deterministic synthetic examples.
- **Lumi for Hermes** dry-run preview adapter.
- Memory-provider compatibility packet examples.
- Module release gates for **Lumi Layered Memory**, **Nuances**, and **Presence**.
- Module export audit, plan, and reviewed-apply tools.
- Deterministic release artifact builder.
- Clean-checkout smoke test.
- Public-readiness audit.
- Public/secret scan.

## Governance boundaries

The v0.1.0 release keeps the self-improvement loop governed through:

- confidence scoring for proposed memory or behavior updates;
- contradiction handling before promotion;
- user approval before durable host-side changes;
- skill evaluation before generated procedures become durable;
- explicit adapter packets instead of silent host memory replacement;
- dry-run/default review cards for **Lumi for Hermes**;
- no private runtime, scheduler, queue, chat, or host identity exposure.

## Lumi for Hermes preview safety

The **Lumi for Hermes** preview is intentionally non-mutating:

```text
canonical_writes: 0
runtime_actions: []
requires_human_review: true
```

It does not read or write Hermes private runtime state, config, memories, scheduler internals, queues, chat metadata, or local profile machinery.
